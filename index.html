#1 Remaferpa

Nosso cliente precisava de um sistema na web para poder gerenciar toda sua empresa, bem como funcionários, registro de pontos (entrada e saída de funcionários), dar acesso aos seus próprios clientes ao sistema financeiro, etc.

O desafio principal foi qual tecnologia escolher, pois precisaríamos trabalhar com comunicação em tempo real para mostrar dados de outros funcionários na tela principal em formato de charts (gráficos), que eram alimentados sempre que uma ordem de serviço era aberta. Tudo isso pra ser mostrado numa TV.

O sistema conta também com permissões dinâmicas de usuários (ACL) e gerenciamento de sessões de usuário em tempo real com Socket.io (o usuário consegue saber onde ele acessou, por qual dispositivo e ainda consegue desconectar todas as sessões pra sua segurança).

Tecnologias utilizadas

Vue.js / Vuex / Vuetify / PWA / Chart.js / Push Notification API / Node.js / MongoDB / Socket.io

#2 Bringuer

Bringuer é um projeto similar ao Grabr, ou seja, pessoas pedem na plataforma enquanto viajantes trazem seus pedidos.

Foi utilizado um template pronto pelo cliente. Ele não quis nada customizado. Algo simples e funcional.

Uma extensão pro Chrome também foi criada para este projeto. Ela pode ser visualizada aqui.

Trata-se de um botão "Comprar com Bringuer" em alguns dos sites aceitos pela plataforma.

Nosso único desafio mesmo foi a necessidade da implementação de um chat na plataforma. Como o PHP não é a melhor linguagem para tal, utilizamos o Laravel como nosso framework para padronizar e agilizar o desenvolvimento. Com isso, foi possível de maneira não muito trabalhosa implementar nosso chat.
